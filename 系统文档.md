## 1. 硬件概况

| 项目 | 规格 |
|------|------|
| 型号 | Raspberry Pi 5 Model B Rev 1.1 |
| CPU | 4 核 ARM Cortex-A76 (aarch64)，BogoMIPS 108.00 |
| 内存 | 7.9 GiB（当前空闲约 3.8 GiB） |
| Swap | 2.0 GiB（zram 压缩交换） |
| 存储 | 59 GiB SD 卡（已用 18 GiB，可用 39 GiB） |

| 接口 | 设备 |
|------|------|

| I²C | `i2c-1`, `i2c-13`, `i2c-14` |
| SPI | `spidev0.1`, `spidev10.0` |
| 串口 | `/dev/ttyAMA0`, `/dev/ttyAMA10` |
| CAN | `can0`（MCP2515 SPI→CAN 模块已加载） |
| 视频 | `/dev/video19` ~ `/dev/video35` |
| USB | 多个 USB 端口（接 USB 声卡、摄像头等） |

## 8. 注意事项

### 8.1 海康 SDK 相关

- 启动前必须设置 `LD_LIBRARY_PATH`，指向 `MakeAll` 和 `MakeAll/HCNetSDKCom` 目录
- 树莓派 ARM64 架构下 `long` 类型为 8 字节，SDK 结构体中使用 `LONG`（实际为 `int` 4字节），修改头文件时需注意
- `plaympeg4.h` 是纯 Windows 头文件，Linux 下需手动声明函数
- `ptz_server` 需要链接 `-lhcnetsdk -lpthread`

### 8.2 串口相关
